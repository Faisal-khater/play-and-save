# play-and-save
Buy &amp; Save (The Turtle’s Journey) is a gamified micro-saving solution within Alinma Bank. It addresses psychological barriers to saving by turning it into an engaging game. Users issue a digital card that automatically rounds up daily purchases (2%-15%), moving a digital turtle across interactive islands to unlock real rewards.
For the User: Overcomes the feeling of financial "deprivation" by replacing it with visual progress and immediate digital rewards, helping users organically build sustainable saving habits without manual effort.  
 For Alinma Bank: Significantly boosts Daily Active Users (DAU) and customer engagement, transforming the app from a utility tool into a lifestyle hub. It also drives higher transaction volumes through the co-branded digital saving card.  
 Strategic Alignment: Fully aligns with Saudi Vision 2030 and the Financial Sector Development Program (FSDP) by actively promoting financial literacy, enhancing the savings rate among Saudi households, and fostering a conscious financial community.
